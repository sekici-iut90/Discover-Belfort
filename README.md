# Discover Belfort – Projet Flutter

Bienvenue sur le dépôt GitHub du projet **Discover Belfort**, réalisé dans le cadre du module Flutter BIP – Module 1.

---

## Description du projet

Ce projet correspond à un **travail pratique Flutter** à rendre pour le module BIP – Module 1.  
L'objectif est de créer une application Flutter fonctionnelle et interactive pour découvrir la ville de Belfort.

---

## Structure du projet

- `lib/` : code source Flutter (Dart)  
- `android/`, `ios/` : configuration spécifique aux plateformes  
- `test/` : fichiers de tests unitaires et widget tests  
- `pubspec.yaml` : gestion des dépendances
